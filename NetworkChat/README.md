# Курсовой проек "Сетевой чат"

## Описание проекта

Разработано два приложения для обмена текстовыми сообщениями по сети с помощью консоли (терминала) между двумя и более пользователями.

Первое приложение - сервер чата, ожидает подключения пользователей. 
Второе приложение - клиент чата, подключается к сервере, умеет отправлять сообщения и получать сообщения через сервер 
от других клиентских приложений. 

Все сообщения записываются в файл log.txt.Выход из чата должен быть осуществлен по команде exit.

## Описание сервера
Подключения клиентов устанавливается через файл настроек settings.txt);
Клиенты могут подключиться к серверу в любой момент и присоединиться к чату;
Новых сообщений отправляются клиентам;
Происходит запись всех отправленных через сервер сообщений с указанием имени пользователя и времени отправки.

## Описание клиента
При подключении предлагается ввести именя для участия в чате;
Настройки для подключения приложения читаются из файла настроек - номер порта и хост сервера;
Для выхода из чата нужно набрать команду выхода - “/exit”;
Каждое сообщение участников записывается в текстовый файл - файл логирования. При каждом запуске приложения файл дополняется.
